# Quiz 2 Review

## Question Breakdown

* File I/O
  * Understand file handles and buffers
  * Be able to read basic file I/O code using Scanner, File, PrintWriter, FileWriter
  * Understand the difference between appending and overwriting
* Arrays
  * Be able to index/set/get values in an array
  * Understand limitations of arrays
  * Understand that arrays are objects
* Multi-Dimensional Arrays
  * Be able to index multidimensional arrays
  * Understand how multidimensional arrays can be used with nested loops
* ArrayLists
  * Understand basic ArrayList methods
  * Understand the benefits of ArrayLists
* Objects
  * Have a basic working understanding of encapsulation
  * Be able to create an object
  * Understand the difference between a class and an object
